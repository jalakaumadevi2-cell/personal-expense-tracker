# Personal Expense Tracker

## 📌 Description
This is a simple Python project that helps users track their expenses and classify spending as Low, Moderate, or High.

## 🚀 Features
- Add multiple expenses
- Calculate total spending
- Categorize expenses
- Simple and beginner-friendly

## 🧠 Concepts Used
- Functions
- Loops
- Conditional Statements

## ▶️ How to Run
1. Install Python
2. Run the program:
   python expense_tracker.py
